# No Hesi 3D - Highway Racing

## Description
A browser-based 3D highway racing game built using HTML, CSS, JavaScript, and the Three.js library. Experience high-speed racing through traffic with futuristic HUD elements and nitro boosts.

## Features
- 3D rendering with Three.js
- Dynamic traffic avoidance
- Nitro boost system
- Distance tracking and high scores
- Futuristic circular HUD

## Controls
- **W**: Gas
- **S**: Brake
- **A / D**: Steer Left / Right
- **SPACE**: Nitro Boost
- **SCROLL**: Zoom Camera In / Out

## Prerequisites
- Node.js installed on your machine.

## How to Run
1. Open up your terminal or command prompt.
2. Navigate to the game directory:
   ```bash
   cd "c:\Users\Hp\OneDrive\Documents\Anti Gravity\NoHesiGame\NoHesi3D"
   ```
3. Start the local server:
   ```bash
   node server.js
   ```
4. Open your web browser and go to:
   ```
   http://127.0.0.1:8080/
   ```
